Pico Sniffer
=================

Projekt "Pico_Sniffer" to sprzętowy sniffer dla magistral UART i I2C bazujący na Raspberry Pi Pico.

Funkcje:

- Przechwytywanie ruchu UART i I2C
- Fuzzing sekwencji transmisji
- Rejestrowanie do CSV oraz przesył przez USB

Zobacz wygenerowaną dokumentację API dla szczegółów modułów i funkcji.
